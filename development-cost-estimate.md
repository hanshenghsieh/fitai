## 💰 API 成本计算

### Claude API 定价（最新）

```
Input:  $3 per 1M tokens
Output: $15 per 1M tokens
```

### 成本分解

```
假设比例：
- Input tokens: 60% (系统提示、已有代码作为上下文)
- Output tokens: 40% (生成的新代码和讨论)

172,000 tokens 总计：
├─ Input: 172,000 × 60% = 103,200 tokens
├─ Output: 172,000 × 40% = 68,800 tokens

计算：
├─ Input: 103,200 × ($3 / 1,000,000) = $0.31
├─ Output: 68,800 × ($15 / 1,000,000) = $1.03
└─ Total: $1.34

保险起见 (考虑迭代):
├─ 低估: $1.34
├─ 中估: $1.34 × 1.5 = $2.01
└─ 高估: $1.34 × 2 = $2.68
```

### 最终 API 成本范围

```
乐观情况:     $1.50
中等情况:     $2.50
保守情况:     $4.00

（包含迭代、调试、修改）
```

```

为什么这么少？
├─ 你已有核心架构 (注册、Onboarding、Dashboard)

```
Claude API: $2-4 (不到 1%)
